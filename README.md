# Mon Blog Personnel

Bienvenue sur le dépôt de mon blog personnel. Ce site est construit avec Jekyll et hébergé sur GitHub Pages.

Vous y trouverez mes articles et réflexions sur divers sujets.

## Structure du site

- Les articles sont dans le dossier `_posts`.
- La configuration du site est dans `_config.yml`.
- La page d'accueil est `index.md`.
